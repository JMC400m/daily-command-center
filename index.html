⚡ One-time setup — syncs Chrome across iPhone, iPad, and desktop

  1. Go to github.com/settings/tokens/new and sign in
  2. Name it daily-command-center · set expiration to No expiration
  3. Under Scopes, check gist only — nothing else needed
  4. Click Generate token and copy it immediately (only shown once)
  5. Open gist.github.com → New gist · filename: dcc-data.json · content: {} · Create secret gist
  6. Copy the Gist ID from the URL — it's the long hex string after your username
  7. Paste both below, click Connect, then bookmark this file on every device
